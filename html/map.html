Overview

This tool can be used to create cloud-free Landsat composites of a study area, perform change detection, create land cover classifications using Collect Earth Online training data, and export the results. The workflow is organized by Inputs and Outputs, each with their own tab.

The Inputs are all of the parameters and settings that the user must specify. These include inputs related to training data, composites, change detection, visualization, and other more advanced settings. Go through each of the sections one at a time, and hover your cursor over the question marks ( ) to read about each input. For more information, see the detailed tutorials in the section below. When you have filled in all of the inputs, click Run.

The Outputs are all of the visualizations, summary data, and export functionality that result from running the tool. These include the maps, change histogram, model accuracy, and exports. They will be available once the the tool has been run. Again, you can hover your cursor over the question marks ( ) to read about each output. For more information, see the detailed tutorials in the section below.
