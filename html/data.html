Collect Earth Online

What

Collect Earth Online is a free and open source web app for collecting training and reference data for geospatial applications, such as land cover classifications. CEO offers simultaneous, cloud supported data entry by multiple users and enables crowd-source visual interpretations of high-resolution satellite and aerial imagery.

In order to create a land cover classification, the Cover Mapper uses something called a supervised classification algorithm (specifically, a random forests model). The “supervised” portion of “supervised classification algorithm” means that you need to teach the algorithm what each category in your classification looks like in the imagery, which you do by providing examples.

Why

In Collect Earth Online, you can easily create a new set of data points, view and interpret high resolution imagery, and provide attributions for those points – and you can do it all in the same place! Unlike many desktop GIS software packages, Collect Earth Online doesn’t require any software installation.

For groups or individuals working outside of the Southern Blues project area, the basic training dataset we’ve provided won’t be of much use – you’ll need to create your own base dataset before getting started with the Cover Mapper.

Who

The average user probably won’t have to interact with Collect Earth Online much (if at all), however each project or study area will likely require a small group of Collect Earth Online users.

In order to create a training dataset for land cover classification with CEO, you’ll need at least one person to set up a project (admin) and three people to collect data (interpreters/analysts) – the admin can be one of the data collectors.
